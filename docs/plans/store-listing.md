## 4. 연령 등급 (제안)
- 의료/건강 정보 취급. 사용자 생성 콘텐츠(댓글) 있음 → 자동 검수 적용.
- 만 14세 미만 가입 차단(온보딩 CHECK). 등급 설문에서 "사용자 간 소통 가능" + "의료/치료 정보" 항목 정직 신고.

## 5. 심사용 데모 계정 (로그인 필수 앱 — 필수 제공)
- 로그인 방식: OAuth(구글·애플·카카오·네이버) + 이메일/비밀번호(`/login`).
- 심사관에게는 **이메일/비밀번호 데모 계정**을 제공(OAuth 불필요).
- 계정 요건: 온보딩 완료(생년월일·성별·약관동의) 상태여야 게이트 통과.
- (작성 예정) 이메일/비번 — Supabase 생성 후 여기 기록.

## 6. 스크린샷 (앱 설치 후 캡처)
- iOS: 6.7"(1290×2796) 또는 6.5". 최소 1~3장.
- Android: 휴대전화 2~8장 + 512 아이콘 + 1024×500 그래픽 이미지.
- 캡처 화면(권장): 홈/검색 · 피부 날씨 · 피부 일기 · 전문의 Q&A 카드.

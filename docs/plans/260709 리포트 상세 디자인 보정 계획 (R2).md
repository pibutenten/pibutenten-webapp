## 1. 원장 지시 → 반영 방안

| 지시 | 반영 |
|---|---|
| 히어로가 첫 화면을 넘음 — 표지처럼 한눈에 | 히어로 총높이 **~750px → 412px(최악치)**. 첫 화면 산수: 헤더 56 + 여백 10 + 히어로 412 = **478px ≤ 가용 640px** — 라벨·시술명·태그·%·인물 그리드·보조문구·헤드라인·저장/공유 전부 표지 안 수납 + 다음 카드 상단이 걸쳐 보여 스크롤 힌트 |
| 인물 숫자·크기 축소 | **100개(10×10, 24px) → 60개(20×3, 9.6px)** — 시안 실측 배열 그대로. 그리드 높이 362→56px. 1칸=1.67%p 라 비율 표현력 유지, 최소 1셀 보장 규칙으로 소수 상태 보존 |

| 세부페이지 헤더: 로고 제거·뒤로 최상단 | AppShell 에 **2뎁스 헤더 variant** 신설 — 좌측 로고 자리에 BackButton, 우측 검색·알림·아바타 유지(시안 원안). 적용: 리포트 상세·프로필(/[handle])·/my/settings. 현행 "로고 헤더 + 아래 뒤로 행" 2줄 → 1줄로 수직 회수. 글상세 등 타 세부페이지 확장은 variant 1줄 부착으로 가능 — 원장 지시 시 일괄 (이번 범위 밖) |

## 2. 작업 항목

### R2-1. 리포트 상세 보정 (ReportsDetailView + procedure-theme)
- **[P1] 히어로 재설계**: 디자이너 A-4 블록표 그대로 (상단 패딩 22 → 라벨 13px → 시술명 34px/800 → 태그 칩 29px → 재시술의향 라벨 → % 68px/800 → 그리드 20×3(gap 6/8) → 보조문구 12.5px·white/85(상향 — 밝아진 배경 대비) → 헤드라인 16px+저장·공유 22px 인라인 → 하단 패딩 22). 그라데이션 `180deg, 원색 0%, 원색 30%, 라이트 140%`. 워터마크 white/10→**/16**. stagger 8→4ms
- **[P1] procedure-theme**: `light` 파생 신설(hue 유지, #B4E4DF 의 S44%·L75% 목표, `#029688→#B4E4DF` 앵커) — `deep` 은 존치(허브 등 타 소비처 무회귀), 히어로 소비만 교체
- **[P2] 하단 패널 #F5FBFF** + 동반 수정: 후기 정렬 칩 sticky 배경 동시 교체 + 비선택 칩 1px 보더 #E1EAF2(대비 보전)
- **[P2] 다른 시술 카드**: 배경 soft(12% 틴트)→`theme.chip` 파스텔, 라운드 24, 높이 ~58, 간격 8
- **[P2] 저장·공유 터치 타깃 44×44**(시각 크기 불변) / **[P2] 통증 라벨 "꽤"→"많이"**(상세 사본)
- **[P3]** 유도 카드 #D6E9F5→#E0F2FB · 작성자 통계 띠 36→30px · 통증/다운타임 마커 26→24px · 하단 고정 바 84→**76px** 밀착+상단 1px 보더 #EFF3F6(배포 후 실기기 겹침 재확인 항목)
- 히어로가 흰 카드 최상단 블록인 현 구조 유지(스크롤 힌트) · 카운트업·비율 로직·배선(§6 체크리스트) 불변

### R2-2. 2뎁스 헤더 variant (AppShell + 3개 배선) — R2-1 과 병렬 (파일 disjoint)
- `AppShell` 에 `backHeader`(가칭) prop: 헤더 좌측 로고+GNB 대신 BackButton(fallbackHref 지정 가능), 우측 아이콘군 유지. 미지정 시 현행 동일. 기존 `back` prop(헤더 아래 행)과 공존 — 이 3화면은 신 variant 로 전환
- 배선: ReportsShell(상세일 때만 — 허브는 로고 유지)·ProfileView(인라인 BackButton 행 제거→헤더로)·MySettingsView(back="/my" 전환)
- 통증 라벨 허브 사본(ReportsIndexCard PAIN_LABELS)과 후기 폼 입력 라벨도 "많이" 로 **3면 통일**(입력↔표시 척도어 일치 — 척도어라 의료광고 가드 무관)

### R2-3. 세부페이지 backHeader 일괄 확장 (원장 추가 확정, 2026-07-09)

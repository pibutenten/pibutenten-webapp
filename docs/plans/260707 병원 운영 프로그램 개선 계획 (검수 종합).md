2. **DB 마이그**: 서브에이전트가 SQL 작성 → 디렉터가 Management API(UTF-8 경로)로 production 적용+재스캔.
3. **전 페이즈 완료 후**: **코드검수관 3인 독립 검수** → 디렉터 종합.
4. **수정사항 있으면**: 디렉터가 **전체 재계획** → 서브에이전트 핀셋 재위임 → **반복**(치명 0 수렴까지).
5. 수렴 후 commit+push(§CLAUDE §3) + 문서 갱신.

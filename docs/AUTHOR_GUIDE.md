# 피부텐텐 의사 답변 작성 가이드 (GEO 강화 + 의료법 컴플라이언스)

> 본 문서는 운영자가 참여 전문의에게 배포하는 작성 가이드입니다.
> 시행: 2026-05-28. 변경 시 docs/CHANGELOG.md 에 기록.

---

## 0. 한 줄 원칙

- 첫 단락은 **40–60자 두괄식 결론**.
- 답변마다 **수치 1개 + PubMed 인용 1개 + 인용 blockquote 1개** 권장.
- 의료법 §56② 금지 표현 (최고·유일·100% 효과·환자 후기·비교·가격) **절대 금지**.

---

## 1. Quick Answer 두괄식 (필수)

답변 첫 단락은 40–60자 자기완결적 결론으로 시작합니다. 이 단락은 자동으로
`.card-answer-speakable` CSS 클래스가 부여되며, AI 음성 비서·답변 엔진이
이 단락을 우선 픽업합니다.

좋음
> "스컬트라는 PLLA(폴리-L-락틱산) 미세입자가 진피에 콜라겐 합성을 유도하는 시술로, 효과는 4-6주에 시작해 24개월까지 유지됩니다."

피해야 할
> "이 질문 정말 좋은 질문이에요. 많은 분들이 궁금해하시는 부분인데요…"

---

## 2. 통계·수치 1개 이상 (Statistics Addition — PAW +30~40%)

Aggarwal et al. (2024, ACM SIGKDD) 통제 실험: 답변에 1개 이상 수치 인용 시 LLM
가시성이 30~40% 상승합니다.

좋음
> "메타분석에서 보툴리눔 톡신 시술 후 6개월 시점 환자 만족도는 평균 78% 였습니다 [Smith 2022, PMID: 33445566]."

피해야 할
> "필러 부작용은 드뭅니다." (수치 없음)

---

## 3. PubMed 인용 1개 이상 (Authoritative References — PAW +30~40%)

본 사이트는 `pubmed_refs` 자동 첨부 기능을 지원합니다. 답변 작성 후 [PubMed
검색] 기능으로 핵심 출처를 1개 이상 첨부해 주세요. DOI 또는 PMID 필수,
최근 5년 이내 출처 70% 이상 권장.

---

## 4. 자기 인용 blockquote 1개 이상 (Quotation Addition — PAW +30~40%)

답변 중 강조 문장을 직접 인용 형식 (`>`) 으로 노출해 주세요. LLM 은
blockquote 를 핵심 인용 후보로 강하게 학습합니다.

좋음
> "필러 후 갑작스러운 피부 변색·심한 통증이 있으면 즉시 시술 받은 병원에 연락해 hyaluronidase 처치를 고려해야 합니다."
> — {작성 전문의 이름}

---

## 5. 회피 표현 사전 (의료법 §56② + AI 광고 필터)

다음 표현은 자동 검수에서 차단되며, LLM 도 인용을 회피합니다.

| 회피 | 위반 조항 | 대안 |
|---|---|---|
| "최고", "최상", "유일", "베스트", "1위" | §56②4호 (비교광고) | "흔히 사용되는", "현재 표준" |
| "100% 효과", "반드시", "확실한", "부작용 없는" | §56②8호 (과장광고) | "○○% 환자에서 보고" + 출처 |
| "OO만의 노하우", "혁신" | §56②4·8호 | 사실 진술 |
| "타 의원보다 우수" | §56②4호 | 의원 비교 표현 일체 삭제 |
| "권위자", "베스트 의사" 등 전문의 간 비교 | §56②4호 | "다양한 관점" 중립 |
| 비급여 가격·할인 정보 | §56②13호 | 가격 정보 일체 게재 금지 |
| "환자분이 ~ 후기를 남겼다" | §56②2호 (치료경험담) | 일반 임상 통계로 환원 |
| "전문병원" (미지정) | §56②9호 | "피부과 전문의가 운영" |
| 인증·표창·수상 광고 | §56②14호 | 공인 인증만 (보건복지부 등) |
| 시술 전후 사진 비교 | §56②6·8호 | 텍스트 위주 설명 |

## 6. 본문 구조

- **H2 = 자연어 질문**: "## 울쎄라 시술 후 욱신거리는 통증이 정상인가요?"
- **H3 = 구체적 sub-topic**
- 한 H2 섹션 ≤ 1500자 (RAG 청크 경계와 의미 경계 일치)
- 문장당 1개 사실, 단락당 1개 핵심 주장
- 비교는 표 (table), 절차는 번호 리스트 (단 셀프 시술 유도 금지)
- 약자는 첫 사용 시 풀어쓰기 — "PLLA (poly-L-lactic acid, 폴리-L-락틱산)"
- 핵심 통계·날짜·약자는 `<strong>` 또는 `<mark>`

---

## 7. 부작용 섹션 의무화 (의료법 §56② 7호)

답변에 시술명이 언급되면 운영자가 관리하는 `procedures` 마스터 DB 에서
부작용·금기를 자동 삽입합니다. 의사가 임상 특수 사례를 추가 기술하는 것은
권장 (의사 자율, 본문 글씨 크기 동일 유지).

본문 글씨 크기와 동일 (14~15px 이상). 작은 disclaimer 글씨는 의협 의료광고
심의위원회 기준상 위반입니다.

---

## 8. 신선도 표기

답변 게재 시 JSON-LD 에 `datePublished` + `dateModified` + `lastReviewed` 가
자동으로 들어갑니다.

본문 마지막에 다음 권장 1줄 추가:
> "본 답변은 [YYYY-MM-DD] 기준 최신 의학 정보로 검수되었습니다."

이유: BrightEdge 데이터 — 60일 내 업데이트 페이지가 AI 답변에 등장할 확률
1.9배.

**⚠ 주의**: 단순 날짜 변경은 Google 2025.12 core update 페널티 대상.
실질적 본문 업데이트가 함께일 때만 갱신해 주세요.

## 9. 재검수 주기

| 주제 | 재검수 주기 |
|---|---|
| 시술·약물 (빠른 변화) | 2년마다 (Mayo Clinic 모델) |
| 일반 피부 상식 | 매년 |
| 안전성 권고·학회 가이드라인 변경 | 즉시 |

last_reviewed_at ≥ 24개월이면 분기별 검수 큐에 자동 진입합니다.

## 10. 환자 사례·임상 관찰 처리

의료법 §56② 2호 (치료경험담 광고 금지) 와 충돌하지 않도록 환자 사례는
**익명·통계화·일반론** 으로 처리합니다.

좋음
> "임상에서 PLLA 시술 후 결절은 마사지 미이행군에서 약 9.3%, 이행군에서 약 1.8% 로 보고되었습니다 [Smith 2022]."

피해야 할
> "30대 여성 A씨가 시술 후 매우 만족하셨습니다."
